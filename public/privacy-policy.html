🛡️ まもるくん

プライバシーポリシー

1. 収集する情報

まもるくんは、以下の情報のみを扱います。

  • 音声データ：録音ボタンを押したときにマイクから取得します。音声認識には、お使いのブラウザ（Google Chrome）に組み込まれた Web Speech API（webkitSpeechRecognition） を使用します。
  • 文字起こしテキスト：音声認識の結果をブラウザのローカルストレージ（chrome.storage.local）に保存します。
  • Gemini API キー：清書機能をご利用の場合、ユーザー自身が入力したAPIキーをローカルストレージに保存します。

2. 情報の使用方法

  • 文字起こしテキストおよびAPIキーは、お使いの端末の外に送信されることはありません。
  • 清書機能（✏️ 清書）を使用した場合、文字起こしテキストを Google Gemini API に直接送信し整形された文章を取得します。この通信はユーザーのブラウザから Google のサーバーへ直接行われ、当拡張機能の開発者を経由しません。

3. Web Speech API について

Chrome の Web Speech API は、音声データを Google のサーバーに送信して文字起こしを行います。これは Chrome ブラウザの機能であり、当拡張機能が制御するものではありません。詳細は Google のプライバシーポリシーをご確認ください。

音声データは当拡張機能の開発者には一切送信されません。Google Chrome と Google の音声認識サービスに対してのみ送信されます。

4. 情報の共有

まもるくんは、ユーザーの情報を第三者に販売・提供・共有しません。

5. データの保持と削除

文字起こしテキストとAPIキーはお使いの端末のみに保存されます。拡張機能の「🗑️ クリア」ボタンでいつでも文字起こしデータを削除できます。APIキーはオプション画面から削除可能です。拡張機能をアンインストールすると、すべてのローカルデータが削除されます。

6. 権限について

当拡張機能が要求する権限は以下のみです。

  • storage：文字起こしテキストとAPIキーをローカルに保存するため
  • generativelanguage.googleapis.com：清書機能でGemini APIを呼び出すため（APIキー設定時のみ）
マイクへのアクセスは拡張機能の宣言権限ではなく、録音ボタンを押したときにブラウザが都度ユーザーへ確認するものです。

7. Google API Limited Use への準拠

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.
