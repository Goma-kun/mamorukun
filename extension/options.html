Gemini API キー（AI清書機能の解放）

✓ APIキー設定済み
🔗 Google Cloud Console でAPIキーを取得する

取得手順：Google Cloud Console にアクセス → プロジェクトを選択または作成 → 「認証情報」→「APIキーを作成」→ コピーして上の欄に貼り付け → 保存。

⚠️ 日本からの利用について：
Google AI Studio（aistudio.google.com）で発行した無料APIキーは、日本からの接続では利用制限がかかる場合があります。
その場合は Google Cloud Console で請求先アカウント（カード登録）を設定した上でAPIキーを発行してください。無料枠内の使用であれば料金は発生しません。

🔒 セキュリティについて：APIキーはお使いの Chrome 内にのみ保存され、外部サーバーへは一切送信しません。ただし以下の点にご注意ください。
・APIキーは他人に共有しないでください
・キーが漏洩した場合は Google AI Studio ですぐに無効化できます
・このデバイスを他の人と共有している場合は使用後にキーを削除することをお勧めします

機能レベルについて

📝 文字起こしのみ（APIキー不要）
リアルタイム音声文字起こし・コピー・クリア。ゼロ設定で使えるメモツールとして機能します。

✨ 文字起こし + AI清書（APIキーあり）
録音停止後に Gemini が自動で清書。えー・あのーなどのフィラー除去、句読点追加、話し言葉の整形を行います。清書タブでいつでも確認・コピー可能。
